0 Acces au panneau d'administration

URL : https://www.ledivino-agde.fr/admin
Connectez-vous avec vos identifiants email / mot de passe.

1 Dashboard (Tableau de bord)

Le Dashboard est la page d'accueil de l'administration. Il affiche en temps reel :

  • Reservations du jour : nombre total, en attente, confirmees, nombre de couverts
  • Reservations de la semaine : total, tendance par rapport a la semaine precedente
  • Taux de confirmation / annulation / no-show
  • Repartition midi / soir sur la semaine
  • Creneau le plus demande
  • Prochaines reservations du jour et de la semaine
  • Prochains evenements
  • Graphique comparatif semaine en cours vs semaine precedente

2 Plats (La Carte)

Consulter les plats

La liste affiche tous les plats avec leur nom, categorie, prix et disponibilite. Filtrez par categorie (Entrees, Plats, Desserts).

Ajouter un plat

Cliquez sur "Ajouter un plat". Remplissez :

  • Nom (francais obligatoire + anglais, italien, espagnol, allemand)
  • Description (francais obligatoire, les autres langues sont generees automatiquement)
  • Categorie : Entree, Plat ou Dessert
  • Source : La Carte (permanente) ou Menu du Marche (plats du jour)
  • Prix en euros
  • Disponibilite : active/desactive

Fonctionnalites IA

  • Generer la description : bouton IA pour creer une description appetissante
  • Traduire automatiquement : traduit le francais vers les 4 autres langues
  • Generer une image : cree une photo du plat par intelligence artificielle

Modifier / Supprimer

Cliquez sur un plat pour le modifier. Le bouton "Supprimer" est disponible dans le formulaire (avec confirmation).

3 Menus (Formules)

Ajouter une formule

  • Nom (multilingue)
  • Type : Entree + Plat, Plat + Dessert, Entree + Plat + Dessert
  • Description (multilingue), Prix, Actif : oui/non

Apres creation, associez des plats a la formule. Seuls les plats disponibles sont proposes.

4 Vins

Ajouter un vin

  • Nom, Region, Appellation, Couleur (Rouge, Blanc, Rose, Petillant)
  • Millesime, Cepage, Degre d'alcool, Style
  • Prix bouteille et Prix au verre
  • Description (multilingue), Image

IA disponible : generation de descriptions de sommelier, traduction automatique, generation d'image.

5 Boissons

Ajouter une boisson

  • Nom (multilingue), Categorie (Softs, Cocktails, Bieres, Spiritueux, Boissons Chaudes)
  • Prix (standard ou multiple : Galopin, 25cl, 50cl, 1L pour les bieres pression)
  • Description (multilingue), Image, Disponibilite

6 Reservations

Modes d'affichage

  • Liste : toutes les reservations avec filtres par statut et date
  • Semaine : vue calendrier hebdomadaire
  • Jour : vue detaillee d'une journee

Statuts

  • En attente (orange) : reserve mais pas confirme par email
  • Confirmee (vert) : le client a clique sur le lien de confirmation
  • Annulee (rouge) : annulee par le client ou le restaurant
  • Terminee (gris) : la visite a eu lieu
  • No-show (gris fonce) : le client n'est pas venu

Actions

  • Changer le statut : cliquez sur une reservation, utilisez le menu deroulant
  • Creer manuellement : bouton "Nouvelle reservation"
  • Supprimer : dans le detail, avec confirmation

7 Clients (Base CRM)

La liste affiche tous les clients ayant interagi avec le site. Recherchez par nom, email ou telephone.

Informations par client

  • Nom complet, email, telephone
  • Nombre de visites et date de derniere visite
  • Allergies connues, preferences alimentaires
  • Historique complet des reservations (date, heure, convives, statut)
La base clients se remplit automatiquement a chaque nouvelle reservation (formulaire du site ou chatbot).

8 Evenements

Ajouter un evenement

  • Titre et Description (multilingue)
  • Type : Karaoke, Concert, Soiree privee, Jour ferie, Animation, Autre
  • Date, Heure de debut / fin, Image

Options de visibilite

  • Mis en avant : affiche en premier sur le site
  • Afficher sur l'ecran : envoye a l'ecran 55"
  • Actif : visible sur le site

IA disponible : generation de contenu, traduction, image, creation d'affiche.

9 FAQ

Ajouter une FAQ

  • Question (francais + 4 langues), Reponse (francais + 4 langues)
  • Ordre d'affichage, Publie : oui/non
Important : Les FAQ sont aussi utilisees par le chatbot comme base de connaissances. Ajoutez-y les questions recurrentes de vos clients.

10 Affiches IA

Creer une affiche

  1. Choisir un template (soiree DJ, karaoke, concert, brunch, happy hour...)
  2. Configurer les variables (nom, date, heure, artiste...)
  3. Choisir l'orientation : Portrait ou Paysage
  4. Choisir la police
  5. Generer : l'IA cree l'image de fond + texte superpose

Actions

  • Telecharger en PNG
  • Envoyer vers l'ecran 55" (push direct)
  • Marquer comme favori
  • Supprimer

11 Galerie

Ajouter des photos

  • Glissez-deposez des images ou cliquez pour parcourir
  • Formats : JPG, PNG, WebP, GIF — Upload multiple possible

Gerer les photos

  • Tag : Restaurant, Plats, Evenements, Equipe, Ambiance
  • Legende (multilingue), Ordre (haut/bas), Publie : visible/masque

12 Ecran 55"

L'ecran du restaurant affiche un diaporama de slides en rotation automatique (8 secondes par slide).

Types de slides

  • Affiche : generee par le module Affiches
  • Image : photo simple
  • Texte : message personnalise
  • Evenement : promotion d'un evenement
  • Menu : affichage d'une formule
  • Plat du jour : mise en avant d'un plat
  • QR Code : QR code du menu digital

Gestion

Activer/Desactiver, reordonner, supprimer. Bouton Apercu pour voir le diaporama en temps reel. Depuis le module Affiches, envoyez directement une affiche vers l'ecran.

13 Parametres

  • Informations restaurant : nom, adresse, telephone, email, site, reseaux sociaux
  • Horaires d'ouverture : par jour, par tranches de 30 minutes
  • Code PIN : acces au mode service
  • QR Code : generation et telechargement (SVG, PNG, PDF en 7x7cm, 10x10cm, A4)

14 Conges (Fermetures exceptionnelles)

  1. Activez le mode conges
  2. Renseignez la date de debut et la date de fin
  3. Ecrivez le message affiche sur le site (multilingue)
Effet : Un bandeau s'affiche sur le site. Le chatbot refuse les reservations pendant cette periode. Desactivez le toggle pour reouvrir.

15 Activite (Journal d'audit)

Toutes les actions sont enregistrees automatiquement :

  • Creation, Modification, Suppression de tout element
  • Filtrable par utilisateur et type d'entite
  • Detail : qui, quoi, quand, quels champs modifies

16 Chatbot Concierge IA

Le chatbot est actif sur le site public. Il fonctionne de maniere autonome avec les donnees de l'administration.

Ce que le chatbot utilise

  • Plats : recommande les plats disponibles avec les prix exacts
  • Vins : conseille des accords mets-vins
  • Boissons : presente la carte des boissons
  • Evenements : informe des evenements a venir
  • Horaires : communique les heures d'ouverture
  • FAQ : repond aux questions frequentes
  • Conges : informe des fermetures exceptionnelles
  • Avis Google : partage la note et les meilleurs avis

Ce que le chatbot fait automatiquement

  • Identifie les clients par prenom/nom et email
  • Prend des reservations (emails de confirmation + alerte WhatsApp)
  • Modifie des reservations (annulation auto + recreation)
  • Annule des reservations (avec email d'annulation)
  • Memorise les preferences et allergies des clients
  • Repond en 5 langues (francais, anglais, italien, espagnol, allemand)

Impact de vos actions

  • Desactivez un plat → le chatbot ne le recommandera plus
  • Ajoutez un evenement → le chatbot le mentionnera
  • Modifiez les horaires → le chatbot utilisera les nouveaux
  • Activez les conges → le chatbot refusera les reservations
  • Ajoutez une FAQ → le chatbot l'utilisera pour repondre

Notifications automatiques

Reservation (site ou chatbot)

  • Email au client : confirmation avec bouton pour valider
  • Email au proprietaire : alerte avec details
  • WhatsApp au proprietaire : notification instantanee

Annulation

  • Email au client : confirmation d'annulation
  • Email + WhatsApp au proprietaire

Modification

  • Email au client : annulation ancienne + confirmation nouvelle
  • Email + WhatsApp au proprietaire

Bonnes pratiques

  1. Mettez a jour les plats regulierement — le chatbot ne recommande que ce qui est disponible
  2. Ajoutez des descriptions en francais — les traductions sont generees automatiquement
  3. Publiez les FAQ importantes — le chatbot les utilise pour repondre aux questions
  4. Verifiez les reservations chaque matin — passez les "terminees" et les "no-show"
  5. Creez les evenements a l'avance — le chatbot les mentionne lors des reservations
  6. Activez les conges avant de partir — le bandeau et le chatbot informent automatiquement
  7. Consultez la page Clients — suivez vos clients fideles et leurs preferences
